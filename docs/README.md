# Garden Solutions – Ordering & Fulfilment System (V1)

## Overview
Garden Solutions is a B2B ordering and fulfilment platform designed for a garden pot manufacturing company. This system facilitates the end-to-end process from sales order capture at nurseries to manufacturing tracking, delivery assignment, and final audit.

## Purpose of this Documentation
This documentation set acts as the authoritative source of truth for the system's architecture, scope, roles, and workflows. It is intended to guide all development and prevent scope drift.
